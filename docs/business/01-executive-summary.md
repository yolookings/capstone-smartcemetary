# Executive Summary

> **Smart Cemetery — Digital Cemetery Management System for Surabaya City**

---

## 1. Business Problem

Traditional cemetery registration in Surabaya involves manual paperwork, in-person visits to government offices, and opaque queuing with no visibility into application status. This creates significant challenges:

- **For Citizens (Families of the Deceased):**
  - Must visit the DLH office multiple times during an emotionally difficult period
  - No way to check application progress without calling or visiting
  - Documents are easily lost or damaged
  - No confirmation that submissions were received

- **For Administrators (DLH Staff):**
  - Paper documents are difficult to organize, search, and archive
  - No centralized view of pending applications
  - Manual allocation of grave locations is slow and error-prone
  - Communication with applicants requires phone calls or physical meetings

- **For the City:**
  - No data on cemetery capacity or utilization rates
  - Difficult to generate reports on burial statistics
  - No audit trail for application decisions

---

## 2. The Solution: Smart Cemetery

Smart Cemetery digitizes the entire burial registration lifecycle, from initial application through document verification to grave allocation and status tracking.

### Core Benefits

| Benefit | Description |
|---------|-------------|
| **24/7 Online Access** | Families can submit burial applications from anywhere, at any time, without visiting the DLH office |
| **Real-Time Status Tracking** | Applicants can check their application status online — no phone calls needed |
| **Digital Document Management** | All required documents (KTP, KK, Surat Kematian) are uploaded and stored securely |
| **Automated Workflow** | Admin review, approval, revision requests, and rejection all happen within the system |
| **Intelligent Grave Allocation** | System automatically assigns the next available plot based on priority order |
| **Multi-Channel Notifications** | WhatsApp and Telegram messages keep both applicants and admins informed of status changes |
| **AI Chatbot Assistant** | 24/7 automated answers to procedural questions about burial regulations |
| **Data-Driven Reporting** | Real-time dashboard with statistics on applications, approvals, and cemetery capacity |
| **Secure & Auditable** | All actions are logged; documents are stored with signed URLs and UUID-based filenames |

---

## 3. Key Stakeholders

| Stakeholder | Role | Needs |
|-------------|------|-------|
| **Citizens / Families** | Submit burial applications, track status | Simple online form, clear status visibility, notifications |
| **DLH Administrators** | Review applications, verify documents, allocate graves | Efficient review interface, document preview, allocation tools |
| **System Administrators** | Manage users, configure cemeteries, generate reports | User management, system configuration, reporting tools |
| **City Management** | Oversee cemetery operations | Utilization statistics, capacity planning data |

---

## 4. Measurable Impact

| Metric | Before | After |
|--------|--------|-------|
| Application time | 1–3 days (in-person) | 10–15 minutes (online) |
| Status inquiry | Phone call or visit | Real-time online dashboard |
| Document management | Physical files | Digital storage with signed URLs |
| Grave allocation | Manual spreadsheet | Automated priority-based system |
| Notification | Manual phone calls | Automated WhatsApp + Telegram |
| Reporting | Manual compilation | Real-time dashboard + export |

---

## 5. Platform Overview

**Live Website:** [https://smartcemetary.web.id](https://smartcemetary.web.id/)

The platform serves two main user types:

1. **Users (Citizens):** Register, submit burial applications, upload documents, track status, receive notifications
2. **Admins (DLH Staff):** Review applications, verify documents, allocate grave plots, manage cemeteries, generate reports
